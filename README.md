# Subliminal Audio PWA Generator

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/leivie312-rgb/Sub.git
   cd Sub
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```

## Development
- To start the development server:
  ```bash
  npm run dev
  ```
- Open `http://localhost:3000` in your browser to see the app in development.

## Build Instructions
- To build the application for production, run:
  ```bash
  npm run build
  ```
- The built files will be generated in the `dist` directory.
